1.1. WHAT IS BLOCKCHAIN?

Blockchain is a decentralized and distributed digital ledger that records transactions across multiple computers or nodes in a secure and transparent manner. It serves as a public database or ledger that maintains a growing list of records called blocks, which are linked together using cryptographic hashes to form a chain. In this all transaction details are sent to all the participants in the blockchain, it shows wide transparency. Here you can thought our transaction detail visible to all, then how it could be secure the answer is that our transaction ID only visible to all participants. Here transaction id meant public key and account address through this our privacy gets protected. There is also a Private key which is a pair of public key, through private key we can transfer money to other in a secure manner, the private key kept secure by the sender. Private keys can be further protected by encrypting them with a passphrase, also known as a password or passphrase encryption. When a passphrase is used, it adds an additional layer of security to the private key, making it more difficult for unauthorized individuals to access and use it. More concepts about "keys" will be in "HOW BLOCKCHAIN WORKS" module.
