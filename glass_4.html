1. Distributed Ledger

A distributed ledger is a type of database that is shared and synchronized across multiple participants or nodes in a network. It provides a decentralized and transparent way to record, store, and update data in a distributed manner.

2.Genesis block

The genesis block is unique because it doesn't reference any previous block since it is the first one. It is typically hardcoded or manually created by the creator of the blockchain network. In Bitcoin, the genesis block created by the pseudonymous creator of Bitcoin, Satoshi Nakamoto, on January 3, 2009.

3. Forks

In a blockchain technology, a fork refers to a divergence or split in the protocol of a blockchain network. It occurs when there is a fundamental change in the rules or consensus mechanism of the blockchain. There are two forks namely hard and soft forks.

4. Hard forks

A hard fork is a permanent divergence in the blockchain where nodes following the old protocol are no longer compatible with the new version. It involves a significant change to the underlying protocol, often introducing new features, rules, or consensus mechanisms. After a hard fork occurs, the blockchain splits into two separate chains, and each chain continues to operate independently with its own set of rules. Participants in the network must upgrade their software to the new version to continue participating in the new chain.

5.Soft Fork

A soft fork is a backward-compatible upgrade to the blockchain protocol. It involves introducing new rules or features that are within the scope of the existing protocol. Nodes running the old software can still participate in the network, but they may not be able to take advantage of the new features or fully validate the new types of transactions. The soft fork typically involves tightening or restricting the rules rather than introducing major changes. Transactions and blocks following the new rules are still recognized by the older nodes, ensuring compatibility and maintaining a single chain.

6. Crypto- currency exchange

A cryptocurrency exchange is a platform or service that allows individuals to buy, sell, and trade various cryptocurrencies. It acts as an intermediary that facilitates the exchange of one cryptocurrency for another or for traditional fiat currencies, such as the US dollar or euro. It's important to research and choose a reputable and well-established cryptocurrency exchange that meets your specific trading needs.

7.Dapps

DApps, short for decentralized applications, are applications that run on a blockchain network.
DApps are designed to operate without the need for a central authority, allowing for transparency, security, and trust among participants. DApps often rely on smart contracts, which are self-executing contracts with the terms of the agreement directly written into code. Smart contracts enable automated and trustless interactions between participants, ensuring that transactions are executed according to predetermined rules without the need for intermediaries.

8.Solidity

Solidity is a programming language specifically designed for developing smart contracts on blockchain platforms, primarily Ethereum. It is a statically-typed, contract-oriented language that allows developers to write code to create and execute smart contracts on the Ethereum Virtual Machine (EVM). It enables developers to create decentralized applications, execute business logic, and interact with blockchain networks in a secure and transparent manner.

9. Non-fungible token (NFT)

NFT stands for Non-Fungible Token. It is a type of digital asset that represents ownership or proof of authenticity of a unique item or piece of content, typically on a blockchain. Unlike cryptocurrencies such as Bitcoin or Ethereum, which are fungible and interchangeable, each NFT has a distinct value and cannot be exchanged on a one-to-one basis.

10. Oracles

An oracle is a mechanism or service that provides external data to the blockchain network or smart contract. Oracles act as bridges between the blockchain and the real world, enabling smart contracts to interact with and make decisions based on data or events that are not directly available on the blockchain. For examples it is used in
Sports Betting :
Oracles can be used in decentralized sports betting platforms to provide real-time scores and outcomes of sporting events. Smart contracts rely on oracles to settle bets and distribute winnings based on the verified results. SportCrypt is an example of a decentralized sports betting platform that uses oracles.

11. MetaMask

MetaMask is a popular cryptocurrency wallet and browser extension that allows users to manage their Ethereum-based assets and interact with decentralized applications (DApps) on the Ethereum network. It provides a user-friendly interface and serves as a bridge between web browsers and the Ethereum blockchain. It enables users to store, send, and receive Ethereum (ETH) and other ERC-20 tokens. It injects a JavaScript library into web pages, enabling DApps to request permission and access the user's wallet for transactions, signing messages, or interacting with smart contracts.

12.Testnets

Ropsten, Rinkeby, and Kovan are three popular test networks or testnets for Ethereum. They are alternative blockchain networks that mimic the behavior of the Ethereum mainnet but are specifically designed for testing and development purposes. Developers and users can deploy and interact with smart contracts and DApps on these testnets without using real Ether (ETH) or incurring costs.
